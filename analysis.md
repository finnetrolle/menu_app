# Анализ проекта

## Структура проекта
Проект представляет собой веб-приложение для расчета пищевой ценности блюд, разработанное на Python с использованием Sanic.

## Текущая архитектура
- Основной файл приложения: `app.py`
- Модели данных находятся в `src/models/`:
  - `ingredient.py` - модель ингредиента
  - `dish.py` - модель блюда
  - `nutrition.py` - модель информации о питательной ценности

## Проблемы и улучшения

### Типизация
Все методы должны иметь аннотации типов для лучшей читаемости и поддержки.
