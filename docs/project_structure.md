# Структура проекта menu_app

## Общая структура

```
menu_app/
├── app.py                 # Основной файл приложения (backend)
├── requirements.txt       # Зависимости Python
├── data/                  # Директория с данными
│   └── ingredients.csv    # CSV файл с ингредиентами
├── dishes/                # Директория с блюдами
├── static/                # Директория со статическими файлами
│   ├── css/               # CSS стили
│   └── js/                # JavaScript файлы
├── templates/             # HTML шаблоны
└── tests/                 # Директория с тестами
    ├── test_data_loading.py  # Тесты загрузки данных
    └── test_app.py          # Тесты приложения
```

## Подробное описание структуры

### 1. app.py
- Основной файл Python приложения
- Реализует логику backend (Sanic web framework)
- Загружает данные из CSV и JSON файлов
- Предоставляет API endpoints для работы с данными

### 2. requirements.txt
- Файл с зависимостями Python
- Содержит список необходимых библиотек для работы приложения

### 3. data/ingredients.csv
- CSV файл с базой ингредиентов
- Содержит информацию о калориях, белках, углеводах и жирах для каждого ингредиента
- Формат: name,energy_kcal,protein_g,carbohydrates_g,fat_g

### 4. dishes/
- Директория с описанием блюд
- Каждое блюдо представлено в отдельном JSON файле
- Каждый файл содержит информацию о названии блюда, его ингредиентах и их количестве в граммах

### 5. static/
- Директория со статическими файлами
- css/ - CSS стили для веб-интерфейса
- js/ - JavaScript файлы для функциональности интерфейса

### 6. templates/
- Директория с HTML шаблонами
- Содержит основной шаблон для отображения приложения

### 7. tests/
- Директория с тестами
- test_data_loading.py - Тесты для проверки корректной загрузки данных
- test_app.py - Unit-тесты для проверки функций расчета

## Использование

### Загрузка данных
1. Ингредиенты загружаются из `data/ingredients.csv`
2. Блюда загружаются из файлов в `dishes/` (каждое блюдо в отдельном JSON файле)

### Добавление новых блюд
1. Создайте новый JSON файл в директории `dishes/`
2. Файл должен содержать поля:
   - name: Название блюда
   - ingredients: Массив ингредиентов с полями name и amount

### Запуск приложения
1. Создайте виртуальное окружение (если не создано): `python -m venv .venv`
2. Активируйте виртуальное окружение:
   - macOS/Linux: `source .venv/bin/activate`
   - Windows: `.venv\Scripts\activate`
3. Установите зависимости: `pip install -r requirements.txt`
4. Запустите сервер: `python app.py`
5. Откройте в браузере: `http://localhost:8000`
